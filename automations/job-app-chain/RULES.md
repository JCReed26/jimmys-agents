# Job App Chain — Rules

_Behavioral rules. Agent-managed._

## Application Criteria
- Apply only to roles matching: software engineer, backend, fullstack, AI/ML
- Minimum salary: $120k (skip if < 100k stated)
- Remote-friendly preferred; on-site OK for Bay Area

## HITL Policy
- Always require HITL approval before submitting any application
- Show full cover letter + resume variant for review
- Allow reject with comment (comment fed back to optimizer)

## Sheet Locking
- Lock Cell A1 (RED) at workflow start, unlock (GREEN) in finally block

## Quality Rules
- Cover letter must reference specific company product/mission
- Resume must tailor experience to job description keywords
- Temperature: 0 for classification, 0 for optimizer
